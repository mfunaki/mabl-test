# 静的HTML生成指示書: index.html

このディレクトリに、アイコン画像とそれを利用した `index.html` を生成してください。

## 1. 画像生成（アイコン）の指示
- このプロジェクトにふさわしい「ドキュメント」または「エディタ」をモチーフにしたアイコン画像を生成してください。
- ファイル名は `icon.png` とし、このディレクトリに保存してください。
- サイズは 512x512 ピクセルの透過PNGとしてください。

## 2. HTMLの基本構成
- 生成した `icon.png` を `<img>` タグで表示し、その横にタイトルを表示してください。
- 言語設定は日本語 (`lang="ja"`)。
- 全体的にモダンなUI（適切な余白、丸みを帯びた角、清潔感のある配色）にしてください。

## 3. UI要素と配置
上から順に以下の要素を配置してください。

1.  **ヘッダーエリア**: 
    - `<img src="icon.png" alt="Icon" width="40" height="40">`
    - `<h1>テキストエディタ・サンプル</h1>`（タイトル）
2.  **メインテキストエリア**: 複数行入力可能な `<textarea>` (ID: `main-input`)
3.  **操作パネル（ボタン群）**:
    - **ボタン1**: 「日本語サンプル1」
    - **ボタン2**: 「日本語サンプル2（全角空白）」
    - **ボタン3**: 「English Sample」
4.  **サブテキストエリア**: メインと同様の `<textarea>` (ID: `sub-input`)
5.  **クリアパネル**:
    - **削除ボタン**: 「下段の内容を全削除」

## 4. 動作仕様（JavaScript）

### サンプル挿入処理（main-inputに対して）
各ボタンクリック時に既存内容を破棄して以下を挿入：
- **ボタン1**: 
  日本語のサンプル。改行を含めて5行程度。
- **ボタン2**: 
  日本語のサンプル。行頭に全角空白を含めて5行程度。
- **ボタン3**: 
  英語のサンプル。半角空白と改行を含めて5行程度。

### 削除処理
- **削除ボタンクリック時**: `sub-input` の内容のみを全削除する。

## 5. 出力に関する注意
- HTMLファイル内では、生成した `icon.png` を正しく参照してください。
- CSS/JSはHTML内にインラインで記述するか、別ファイルにする場合は適切にリンクさせてください。
