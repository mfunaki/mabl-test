# mabl-test - 日付・時刻入力テスト

## テスト概要
- **対象URL**: {app.url}/011/
- **テスト名**: mabl-test - 日付・時刻入力テスト
- **目的**: 日付入力（カレンダー、リストボックス、ウィジェット）および時刻入力（リストボックス、ウィジェット）の各入力方法が正しく動作し、選択結果が表示されることを確認する
- **ラベル**: `ai-generated`, `form`

## 画面構成

ページは大きく「日付入力」セクションと「時刻入力」セクションの2つに分かれている。

### 日付入力セクション

1. **「1. カレンダーから日付を設定」** — 「日付:」ラベルの横にブラウザ標準の日付入力フィールドがある。日付を選択すると、その下の結果エリアに「○○月○○日が指定されました」と表示される。
2. **「2. 月・日をリストボックスから選択」** — 「月:」と「日:」の2つのリストボックス（ドロップダウン）がある。初期値はどちらも「--」。月を選択すると、日のリストボックスの選択肢がその月の日数に合わせて更新される。両方を選択すると、その下の結果エリアに「○○月○○日が指定されました」と表示される。
3. **「3. ウィジェットから月・日を選択」** — 青いヘッダーに月名と前月（«）・次月（»）ボタンがあるカレンダーウィジェット。曜日ヘッダー（日〜土）の下に日付のセルが並ぶ。日曜は赤文字、土曜は青文字で表示される。日付セルをクリックすると青い背景で選択状態になり、その下の結果エリアに「○○月○○日が指定されました」と表示される。

### 時刻入力セクション

1. **「1. 時・分をリストボックスから選択」** — 「時:」と「分:」の2つのリストボックスがある。初期値はどちらも「--」。時は00時〜23時、分は00分〜59分。両方を選択すると、その下の結果エリアに「○○時○○分が指定されました」と表示される。
2. **「2. ウィジェットから時・分を選択」** — 青いヘッダーに「時」「分」と表示された2列のスクロール可能なリストがあり、間に「:」の区切り文字がある。時の列は00〜23、分の列は00〜59が並ぶ。それぞれクリックすると青い背景で選択状態になり、選択した要素が見える位置にスクロールする。両方を選択すると、その下の結果エリアに「○○時○○分が指定されました」と表示される。

### その他

- ページ下部に「← トップページに戻る」リンクがある。
- 各結果エリアは未選択時に「（未選択）」と薄いグレーで表示される。

## テスト手順

### 1. ページ表示の確認

1. {app.url}/011/ に直接アクセスする
2. ページタイトル「日付・時刻入力」が表示されていることを確認する
3. 「日付入力」セクションが表示されていることを確認する
4. 「時刻入力」セクションが表示されていることを確認する

### 2. カレンダーから日付を設定

5. 「日付入力」セクションの「1. カレンダーから日付を設定」にある日付入力フィールドに「2025-03-15」を入力する
6. 日付入力フィールドの下の結果エリアに「03月15日が指定されました」と表示されていることを確認する

### 3. 月・日をリストボックスから選択

7. 「2. 月・日をリストボックスから選択」にある「月:」リストボックスから「7月」を選択する
8. 「日:」リストボックスから「20日」を選択する
9. リストボックスの下の結果エリアに「07月20日が指定されました」と表示されていることを確認する

### 4. 月の変更による日の選択肢の更新を確認

10. 「月:」リストボックスから「2月」を選択する
11. 「日:」リストボックスの選択肢が最大29日までになっていることを確認する（2月は29日まで）
12. 「日:」リストボックスから「29日」を選択する
13. リストボックスの下の結果エリアに「02月29日が指定されました」と表示されていることを確認する

### 5. カレンダーウィジェットから日付を選択

14. 「3. ウィジェットから月・日を選択」のカレンダーウィジェットで、次月ボタン（»）をクリックして月を1つ進める
15. ヘッダーの月表示が変わったことを確認する
16. 前月ボタン（«）をクリックして月を1つ戻す
17. ヘッダーの月表示が元に戻ったことを確認する
18. カレンダーウィジェット内の「10」の日付セルをクリックする
19. クリックした日付セルが青い背景の選択状態になっていることを確認する
20. カレンダーウィジェットの下の結果エリアに「○○月10日が指定されました」と表示されていることを確認する

### 6. 時・分をリストボックスから選択

21. 「時刻入力」セクションの「1. 時・分をリストボックスから選択」にある「時:」リストボックスから「14時」を選択する
22. 「分:」リストボックスから「30分」を選択する
23. リストボックスの下の結果エリアに「14時30分が指定されました」と表示されていることを確認する

### 7. 時刻ウィジェットから時・分を選択

24. 「2. ウィジェットから時・分を選択」の時刻ウィジェットで、「時」列から「09」をクリックする
25. 「分」列から「45」をクリックする
26. 時刻ウィジェットの下の結果エリアに「09時45分が指定されました」と表示されていることを確認する

### 8. トップページへの遷移確認

27. ページ下部の「← トップページに戻る」リンクをクリックする
28. トップページに遷移し、「ようこそ mabl-test へ」という見出しが表示されていることを確認する

## 期待される結果

- カレンダー入力フィールドで日付を設定すると、結果エリアに「○○月○○日が指定されました」と表示される
- 月・日のリストボックスで両方を選択すると、結果エリアに「○○月○○日が指定されました」と表示される
- 月のリストボックスを変更すると、日のリストボックスの選択肢がその月の日数に合わせて更新される
- カレンダーウィジェットの前月・次月ボタンで月が切り替わり、日付セルをクリックすると選択状態になり結果が表示される
- 時・分のリストボックスで両方を選択すると、結果エリアに「○○時○○分が指定されました」と表示される
- 時刻ウィジェットで時と分をクリックすると選択状態になり結果が表示される
- 「← トップページに戻る」リンクからトップページに遷移できる

## 注意事項

- カレンダーウィジェットの初期表示月は現在の月であるため、テスト実行時期によってウィジェットの表示内容が異なる。ステップ20の月の値はテスト実行時の月に依存する
- ブラウザ標準の日付入力フィールド（type="date"）の表示形式やカレンダーUIはブラウザによって異なるため、日付の入力方法がブラウザに依存する
- このページにはlocalStorageやサーバーサイドの状態保存はないため、ページをリロードするとすべての選択状態がリセットされる
- 時刻ウィジェットの「分」列は00〜59の60項目があり、スクロールが必要になる。mablがスクロール後にクリックできることを確認する
